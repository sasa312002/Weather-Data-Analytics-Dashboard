# 🌦️ Weather Data Analytics - Power BI Dashboard

## 📌 Project Overview
This project features an interactive Power BI dashboard designed to analyze and visualize meteorological data. The dashboard provides deep insights into historical weather trends, regional climate variations, and key environmental metrics. It was built to demonstrate end-to-end business intelligence capabilities, from data extraction and modeling to advanced visualization.

## 📊 Key Features
* **Interactive Visualizations:** Dynamic charts and graphs detailing temperature fluctuations, humidity levels, and precipitation over time.
* **Geospatial Mapping:** Interactive maps that visualize weather patterns across different geographic regions.
* **Advanced DAX Measures:** Custom calculations for time-intelligence, enabling year-over-year and month-over-month trend comparisons.
* **Optimized Data Model:** Built on a robust star schema using Power Query for efficient ETL (Extract, Transform, Load) processing.
* **Dynamic Filtering:** Slicers and drill-through capabilities allowing users to filter data by date range, location, and specific weather conditions.
